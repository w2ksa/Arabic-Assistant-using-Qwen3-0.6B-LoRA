# Portfolio Notes

Repository: Arabic-Assistant-using-Qwen3-0.6B-LoRA

CV bullet:
Developed a general-purpose Arabic AI assistant prototype using Qwen3-0.6B, Python, Hugging Face Transformers, and Gradio, with local inference, Arabic prompt formatting, sample instruction data, model attribution, and a LoRA fine-tuning-ready structure.

Technical CV bullet:
Built an Arabic LLM adaptation project on top of the open-source Qwen3-0.6B model, including a Python inference pipeline, Gradio web interface, sample Arabic instruction dataset, evaluation documentation, and a planned LoRA fine-tuning workflow for progressive model improvement.

Interview explanation:
The project does not train a foundation model from scratch. It uses Qwen3-0.6B as an open-source base model and focuses on practical AI engineering: running the model locally, building an inference layer, adding an interface, preparing Arabic data, documenting attribution, and preparing the repository for LoRA fine-tuning.

Current status:
Working prototype and fine-tuning-ready project structure. LoRA training is planned as the next phase after local testing and dataset expansion.
