# Evaluation Notes

This document tracks model behavior before and after future LoRA fine-tuning.

## Evaluation Method

Use the same prompts before and after training.

For each prompt, compare:

- Arabic fluency
- Accuracy
- Clarity
- Ability to follow instructions
- Practical usefulness
- Consistency
